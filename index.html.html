Firebase Live Sync Setup

To enable live data sync across all devices, you need a free Firebase project. Follow these steps:

Step 1: Create Firebase Project

Go to console.firebase.google.com → Click "Add project" → Enter any name → Create

Step 2: Create Realtime Database

In your project → Build → Realtime Database → Create Database → Choose region → Start in test mode

Step 3: Get Your Config

Go to Project Settings (gear icon) → General → Your apps → Click Web (</>) → Register app → Copy the config values below

Incentive Rules Applied (Auto-Calculated)

GP & Qty Incentive Slabs:

≥ 95% → 5% on GP + 5% on Qty
≥ 90% < 95% → 4% on GP + 4% on Qty
≥ 80% < 90% → 3% on GP + 3% on Qty
< 80% → No Incentive
Mandatory Conditions:

• Min GP achievement: 80%+
• Full GP incentive: needs 100% Samsung Val target
• Full Qty incentive: needs ≥15% Ageing of qty target
• BTD = Balance To Date = Target − Achieved
• Per Day Req = BTD ÷ Remaining working days
